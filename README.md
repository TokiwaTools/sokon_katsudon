### 活動教室とは
- 総コンの活動のため普段確保している教室
- 中野キャンパスには部室がないので、授業で使う教室を利用している
- なぜか ==カツ丼== ==かっきょ== とも言われている

### 利用期間
- 平日 **2限** (10:50) ～ **夜** (20:50)
- **教室の空き状況によっては取れない時間帯もある**

---

### 利用目的
- 制作活動
- メンバーとの交流
- 空き時間の居場所
- (昼飯食べるとこ)

---

### ルール
- 利用時間帯を守る
- 出かけるときは貴重品を置いていかない
- 最後に教室を出る人は
	- 椅子や机を元の状態に戻す
	- 荷物は持ち帰る
	- 電気、エアコン、黒板の文字を消す
